# Система автоматизированного проведения и оценки голосовых ролевых кейсов

## О проекте

Репозиторий содержит прототип системы автоматизированного проведения и оценки голосовых ролевых кейсов для подбора специалистов контактных центров. Проект выполнен в рамках выпускной квалификационной работы и посвящён применению речевых технологий и больших языковых моделей в задачах оценки кандидатов.

## Цель работы

Целью выпускной квалификационной работы является разработка и апробация прототипа системы автоматизированного проведения и оценки голосовых ролевых кейсов для подбора специалистов контактных центров на базе речевых и больших языковых моделей.

## Задачи

В рамках работы решаются следующие задачи:

1. Спроектировать архитектуру и программно реализовать цифровой продукт.
2. Интегрировать сервисы распознавания и синтеза речи.
3. Разработать алгоритм автоматизированного скоринга кандидатов на базе LLM.
4. Провести апробацию системы на тестовой выборке диалогов.
5. Оценить согласованность результатов AI-оценивания с экспертными оценками HR-специалистов.

## Функциональные возможности

Текущая версия прототипа поддерживает:

- проведение голосовых ролевых кейсов;
- распознавание и обработку речевых реплик пользователя (ASR);
- генерацию реплик системы (TTS);
- автоматизированный скоринг ответов кандидата на базе LLM;
- сохранение и структурирование результатов для последующего анализа;
- взаимодействие клиентской и серверной частей приложения через REST API.

## Структура репозитория

- `attached_assets/` — сценарии ролевых кейсов и вспомогательные материалы для апробации;
- `backend/` — серверная логика и обработка данных;
- `frontend/` — клиентская часть приложения;
- `lib/` — общие модули проекта: API-клиент, спецификации API, Zod-схемы, модели базы данных;
- `scripts/` — служебные скрипты;
- `main.py` — точка входа;
- `package.json` — конфигурация JavaScript-части проекта;
- `pnpm-lock.yaml` — lock-файл зависимостей;
- `pnpm-workspace.yaml` — конфигурация workspace;
- `pyproject.toml` — конфигурация Python-части проекта;
- `.gitignore`, `.replit`, `replit.nix`, `replit.md` — файлы окружения и настройки проекта.
